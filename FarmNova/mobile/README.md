# FarmNova Mobile App

Welcome to the **FarmNova** mobile application! This app is designed to connect farmers directly to consumers, providing a seamless experience for browsing products, managing subscriptions, and engaging in real-time chat.

## Features

- **User Authentication**: Secure login and registration for farmers and consumers.
- **Product Browsing**: Easily browse through a variety of fresh farm produce.
- **Weekly Subscription Box**: Subscribe to receive curated boxes of fresh produce weekly.
- **Order Management**: Track your orders and manage subscriptions effortlessly.
- **Real-Time Chat**: Engage with farmers and consumers through our integrated chat feature.
- **Weather Updates**: Get live weather information based on your location.

## Getting Started

To get started with the FarmNova mobile app, follow these steps:

1. **Clone the Repository**:
   ```
   git clone https://github.com/yourusername/FarmNova.git
   ```

2. **Navigate to the Mobile Directory**:
   ```
   cd FarmNova/mobile
   ```

3. **Install Dependencies**:
   ```
   npm install
   ```

4. **Run the Application**:
   ```
   npm start
   ```
