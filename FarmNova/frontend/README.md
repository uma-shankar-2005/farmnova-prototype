# FarmNova Frontend

Welcome to the FarmNova frontend! This project is built using the MERN stack (MongoDB, Express.js, React.js, Node.js) and aims to connect farmers directly to consumers, facilitating both B2B and B2C transactions.

## Getting Started

To get started with the FarmNova frontend, follow these steps:

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/FarmNova.git
   ```

2. Navigate to the frontend directory:
   ```
   cd FarmNova/frontend
   ```

3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To run the application in development mode, use the following command:
```
npm start
```
This will start the React application and open it in your default web browser.

### Building for Production

To create a production build of the application, run:
```
npm run build
```
The build artifacts will be stored in the `build` directory.

## Features

- **User Authentication**: Secure login and registration for farmers and consumers.
- **Product Listings**: Browse and search for farm products.
- **Subscription Box System**: Subscribe to receive curated boxes of fresh produce weekly.
- **Real-Time Market Prices**: View live mandi prices and price trends.
- **Weather Reports**: Get live weather data based on your location.
- **AI-Powered Price Suggestions**: Smart pricing suggestions for farmers.
- **Chat Support**: Real-time chat support for users.

## Folder Structure

- `public/`: Contains the main HTML file and static assets.
- `src/`: Contains the main application code.
  - `assets/`: Static assets like images and fonts.
  - `components/`: Reusable components.
  - `contexts/`: Context providers for state management.
  - `hooks/`: Custom hooks for reusable logic.
  - `pages/`: Main pages of the application.
  - `services/`: Functions to interact with the backend API.
  - `styles/`: CSS styles for the application.
